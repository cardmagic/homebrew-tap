# Homebrew Tap

Personal Homebrew tap for cardmagic packages.

## Installation

```bash
brew tap cardmagic/tap
```

## Packages

### messages

Fuzzy search and browse Apple Messages/iMessage from CLI.

```bash
brew install cardmagic/tap/messages
```

Or directly:

```bash
brew install cardmagic/tap/messages
```

**Note:** Requires Full Disk Access to read Apple Messages database.

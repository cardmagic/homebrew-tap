# Homebrew Tap

Personal Homebrew tap for cardmagic packages.

## Installation

```bash
brew tap cardmagic/tap
```

## Packages

| Formula | Description | Install |
|---------|-------------|---------|
| [classifier](https://rubyclassifier.com) | Text classification with Bayesian, LSI, Logistic Regression, and kNN | `brew install cardmagic/tap/classifier` |
| [messages](https://github.com/cardmagic/messages) | Fuzzy search and browse Apple Messages/iMessage from CLI | `brew install cardmagic/tap/messages` |
| [notes](https://github.com/cardmagic/notes) | Fuzzy search, browse, create, and delete Apple Notes from CLI | `brew install cardmagic/tap/notes` |
| [reminders](https://github.com/cardmagic/reminders) | Fuzzy search and browse Apple Reminders from CLI | `brew install cardmagic/tap/reminders` |

## Requirements

- **messages**, **notes**, and **reminders** require Full Disk Access (System Settings > Privacy & Security > Full Disk Access)
